# Repository content

This repository contains materials used for Multiple Regression (Lectures 1-4) in Behavioral Research Methods (BRM) 2 and 3.
Lectures 1-3 are part of BRM2 and Lecture 4 is part of BRM3.

# Repository structure

``` markdown
multiple_regression/
├─ lectures/
|   ├─ lecture_1.qmd: Quarto file containing content, code and data to generate "lecture_1.hmtl" (multiple-regression 1).
|   ├─ lecture_2.qmd: Quarto file containing content, code and data to generate "lecture_2.hmtl" (multiple-regression 2).
|   ├─ lecture_3.qmd: Quarto file containing content, code and data to generate "lecture_3.hmtl" (multiple-regression 3).
|   └─ lecture_4.qmd: Quarto file containing content, code and data to generate "lecture_4.hmtl" (multiple-regression 4).
└─ data/grades.csv: Raw dataset used in "multiple_regression.qmd".
└─ _book/
    ├─ index.html: HTML book generated by rendering all .qmd files located in the multiple_regression/lectures/.
    └─ lectures/: a folder with all individual lectures in .html format.
```

# How to use this project:

Download the project, open the R project, then open index.qmd and click Render. 
This will open the Quarto book in your browser, where you can navigate through the chapters.
