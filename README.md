# Rpository content

This repository contains materials used for Multiple Regression (Lectures 1-4) in Behavioral Research Methods (BRM) 2 and 3

# Repository structure

```markdown
multiple_regression/
├─ lectures/
|   ├─ lecture_1.qmd: Quarto file containing content, code and data to generate "lecture_1.hmtl"
|   ├─ lecture_2.qmd: Quarto file containing content, code and data to generate "lecture_2.hmtl"
|   ├─ lecture_3.qmd: Quarto file containing content, code and data to generate "lecture_3.hmtl"
|   └─ lecture_4.qmd: Quarto file containing content, code and data to generate "lecture_4.hmtl"
└─ data/grades.csv: Raw dataset used in "multiple_regression.qmd".
└─ _book/
    ├─ index.html: HTML output generated by rendering the .qmd files located in the folder lectures/.
    └─ lectures/: a folder with all individual lectures in .html format.
```
